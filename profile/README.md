# Readmigo

**AI-Powered English Reading Companion**

Readmigo helps language learners improve their English through immersive reading experiences with AI-assisted comprehension tools.

## Features

- **Smart Reading** - Read classic literature with AI-powered word explanations, sentence simplification, and paragraph translations
- **Vocabulary Building** - Build your vocabulary with spaced repetition and context-based learning
- **Progress Tracking** - Track reading progress, learning statistics, and achievement badges
- **Text-to-Speech** - Listen to content with natural voice synthesis
- **Bilingual Support** - Side-by-side English-Chinese reading for select titles
